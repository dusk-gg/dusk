# Outmatched

A multiplayer Match 3 game using the Dusk SDK. Here's a [demo](https://developers.dusk.gg/examples/outmatched/).

[<img src="../../docs/static/img/multiplayer-games/outmatched.png" width=500>](https://developers.dusk.gg/examples/outmatched/)

## Installing

```sh
yarn
```

## Running

```sh
yarn dev
```
