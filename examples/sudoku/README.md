# Sudoku

A multiplayer Sudoku game using the Rune SDK.

<img width="761" alt="Screenshot of Sudoku" src="https://user-images.githubusercontent.com/7106681/239557490-beb340c3-4bc9-4633-b3ea-f59eebcd223e.png">

## Install

```sh
yarn
```

## Run for development

```sh
yarn start
```

## Build for upload

```sh
yarn build
```
