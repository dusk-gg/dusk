# Pinpoint

A multiplayer Pinpoint game using the Rune SDK.

<img width="761" alt="Screenshot of Pinpoint" src="https://github.com/rune/rune-games-sdk/assets/7106681/c867663f-ed0d-4e72-a150-b0b62a6f2d41">

## Install

```sh
yarn
```

## Run for development

```sh
yarn start
```

## Build for upload

```sh
yarn build
```
